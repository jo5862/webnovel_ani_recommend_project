## 📖 요약

### 1.개요

- aniflIix는 애니메이션, 웹소설 사이트 인기 컨텐츠 데이터를 수집하고 분류한 데이터를 통해 이용자가 검색한 키워드에 맞춰 이와 유사한 인기 컨텐츠를 추천해줘 사이트 구독율 유지에 활용 될 것을 목표로 합니다.

### 2.문제의식

- 애니메이션, 웹소설 사이트 무료 구독기간에서 유료 구독기간으로 넘어가기 전 구독 해지 이유 중 선호 작품을 추가로 탐색하기 어려워 사이트를 이용하지 않아 구독 해지를 하는 점 확인했습니다.

### 3.모델 학습 및 최적화(TF-IDF,Word2Vec)

**키워드:** TfidfVectorizer, sublinear_tf, TF-IDF 매트릭스 생성, 피클 저장, 희소 행렬 저장, 데이터 벡터화.

**요약:**

TfidfVectorizer를 사용하여 전처리된 리뷰 데이터를 TF-IDF 방식으로 벡터화하고, 생성된 TF-IDF 모델과 행렬을 각각 피클 파일과 희소 행렬 형식으로 저장했습니다.

**키워드:** Word2Vec, 토큰화, 임베딩 모델, Word2Vec 모델 학습, 모델 저장, 단어 임베딩.

**요약:**

리뷰 데이터를 공백을 기준으로 토큰화한 후, Word2Vec 모델을 학습하여 단어 임베딩을 생성하고, 학습된 모델을 파일로 저장했습니다.

### 4.기능 구현

**키워드:** 추천 시스템, 코사인 유사도, Tfidf, Word2Vec, 유사 단어 추출, 영화 추천

**요약:**

사용자가 입력한 키워드와 유사한 단어를 Word2Vec 모델로 찾고, 해당 단어들을 조합하여 생성한 문장에 대해 Tfidf 기반 코사인 유사도를 계산하여 영화 추천 목록을 제공합니다.

**키워드:** 추천 시스템, 코사인 유사도, Tfidf, Word2Vec, 유사 단어 추출, 영화 추천

**요약:**

사용자가 입력한 키워드와 유사한 단어를 Word2Vec 모델로 찾고, 해당 단어들을 조합하여 생성한 문장에 대해 Tfidf 기반 코사인 유사도를 계산하여 영화 추천 목록을 제공합니다.

- 처음에는 리디북스에서 만화 데이터를 수집하는 것을 목표로잡았으나
- 만화데이터의 경우 형태소를 분리를 함에 있어 적절하지 않고 질적으로 좋지 않아 다른 데이터를 수집하는 것으로 방향을 바꿈
- 웹소설 독자 리뷰데이터의 경우 컨텐츠 이용 시간 및 이해에 있어 만화에 비해 시간이 걸리기 때문에 이에 대한 리뷰 데이터의 신뢰도가 높은 것을 확인했음
- 애니메이션의 경우 라프텔에서 진행했으며 데이터의 형태소 분류에 있어 질적으로 좋지 않은 것을 동일하게 느꼇으나 양적인 부분에 있어서는 충분하기 때문에 채택했음
- 데이터 양은 애니 총 191개, 웹소설 총 515개(로맨스 190개,  로판 165개, 판타지 160개)입니다.
- 애니 리뷰 개수총 9081개, 웹소설 리뷰 총 23,029개로 총 리뷰 데이터는  32,110개로 작업했습니다.

이 코드는 PyQt5를 사용한 GUI 기반 웹소설 추천 시스템입니다. 사용자에게 영화 또는 웹소설을 추천하는 시스템으로, 추천은 사용자가 입력한 키워드나 제목을 바탕으로 Tfidf와 Word2Vec 모델을 활용하여 추천합니다. 주요 흐름은 다음과 같습니다:

1. **PyQt5 UI 설정**:
    - `QComboBox`에 웹소설 제목을 추가하고, `QLineEdit`에 키워드를 입력받습니다.
    - 추천 버튼 클릭 시, 키워드 또는 제목을 바탕으로 추천을 처리합니다.
2. **추천 시스템**:
    - **제목 기반 추천**: 선택된 제목을 바탕으로 Tfidf 행렬을 사용하여 유사한 제목을 추천합니다.
    - **키워드 기반 추천**: 입력된 키워드에 대해 Word2Vec을 사용하여 유사 단어를 찾고, 해당 단어들을 사용하여 Tfidf로 유사도를 계산하여 추천을 생성합니다.
3. **추천 결과 표시**:
    - `QLabel`에 추천된 웹소설 목록을 각기 다른 링크 형태로 표시합니다. 각 추천 항목은 `Laftel`과 `RidiBooks`에서 검색할 수 있도록 링크가 생성됩니다.
4. **이미지 표시**:
    - 라벨에 이미지를 표시하여 UI를 다채롭게 구성합니다.

### 주요 기능:

- **자동 완성**: 제목 검색 시 자동 완성 기능 제공.
- **추천 시스템**: Tfidf와 Word2Vec을 결합하여 제목 또는 키워드에 대한 추천.
- **웹소설 사이트 링크 제공**: 추천된 웹소설에 대해 `Laftel`과 `RidiBooks` 링크를 제공.

### 동작 순서:

1. 사용자 입력: 제목 선택 또는 키워드 입력.
2. 추천 버튼 클릭 → 추천 처리 (제목/키워드 기반).
3. 추천된 목록을 웹소설 사이트 링크로 표시.

### 4.주요 구현 기능

- 수집한 애니메이션,웹소설 제목을 콤보박스에 기록해두어 자기가 원하는 제목의 작품을 클릭하면 이와 유사한 작품을 애니메이션 5개 작품, 웹소설 5개 작품을 노출 시킴
- 자기가 좋아하는 주제를 직접 입력 하여 추천 버튼을 눌러주면 이와 연관성 높은 작품을 애니메이션 5개 작품, 웹소설 5개 작품을 노출 시킴
- 앞에서 노출된 각 작품들 중 마음에 드는 작품을 마우스로 클릭하게 되면 웹소설에 위치한 작품은 리디 웹페이지가 팝업되며 해당 작품 정보가 노출, 애니메이션의 경우 라프텔 웹페이지가 팝업되며 해당 작품 정보가 노출
